Apresentações¶

Leia mais neste capítulo: apresentações de cm.

Aqui estão alguns sites que oferecem recursos gratuitos fazendo diferentes aspectos da presença:

  • O Noun Project para ícones e fotos.

  • Unsplash para fotos.

  • Pexels para fotos.

  • Freepik para ícones e ilustrações.

  • Flaticon para ícones.

  • Cancele para ícones.

Para gravar palestras:

  • OBS. Consulte este guia para usar o OBS pelo Instituto de Sustentabilidade do Programa.

  • Amara ou Otter podem ser usados para gerar suas legendas. Plataformas de videoconferência, como Zoom ou Google meet também possuem legendas.

  • Muitos dos aspectos técnicos e ferramentas para dar palestras remotas (ou aulas) são cobertas pelo As dicas de Carpintarias para ensinar online e o webinar "Ensinando Online na Breve Aviso" pelo RStudio, uma fonte para vários pontos desta seção.

  • A conferência do Outlier tem algumas excelentes dicas sobre desenvolvimento de fala, estruturae configuração técnica.

  • O autor Nancy Duarte oferece algumas informações sobre seus livros, artigos e palestras sobre como estruturar apresentações que envolvem chamadas à ação.
